# AddisConnect
### Connecting local services to people, instantly.

## 📌 Project Description
AddisConnect is a mobile application designed to solve the problem of finding reliable local service providers in Ethiopia. Users can easily search, view, and connect with verified professionals such as electricians, plumbers, and delivery personnel.

This platform improves efficiency, reduces dependency on word-of-mouth, and builds trust through ratings and profiles.
